## 1. 題目名稱:
* minesweeper踩地雷

## 2. 一句話描述這個 project 在做什麼:
* 模仿與window一模一樣的踩地雷遊戲

## 3. Deployed 連結:
* https://aabb15768.github.io/midterm_minesweeper

## 4. 使用/操作方式:
* 右鍵放置旗子標示你認為為地雷的地方
* 左鍵點擊你認為不是地雷的地方
* 上面兩個計時器中間的笑臉/哭臉，可以重置遊戲
* 最下方的bar可以選擇難度，跟window一樣共有三種
* 當標示出全部的地雷時，遊戲為win
* 當左鍵點擊到地雷時，game over
